# Excel数据脱敏工具 - 快速开始

## 方式一：直接运行（需要 Python）

如果你电脑上安装了 Python：

```bash
# 安装依赖
pip install -r requirements.txt

# 启动 Web 界面
python app.py
```

然后浏览器访问 http://localhost:5000

---

## 方式二：使用打包好的 Windows EXE（无需 Python）

### 如何获取 exe 文件？

由于你目前没有 Windows 环境，最简单的方法是使用 **GitHub Actions 自动打包**：

#### 步骤 1：推送代码到 GitHub

```bash
# 在项目中运行
bash init-git.sh
```

或直接手动操作：
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/你的用户名/你的仓库名.git
git push -u origin main
```

#### 步骤 2：下载打包好的 exe

推送代码后，GitHub Actions 会自动在 Windows 环境下打包：

1. 访问你的 GitHub 仓库
2. 点击 "Actions" 标签页
3. 等待 "Build Windows EXE" 工作流完成（约 2-3 分钟）
4. 点击进入最新的一次运行
5. 页面底部的 "Artifacts" 区域下载 `Excel数据脱敏工具-Windows`

#### 步骤 3：使用 exe

下载解压后，双击 `Excel数据脱敏工具.exe`：
- ✅ 自动打开浏览器
- ✅ 无需安装 Python
- ✅ 无需安装任何依赖

## 功能使用

### 数据脱敏
1. 切换到"数据脱敏"标签
2. 上传 Excel 文件
3. 勾选需要脱敏的列（如：姓名、手机号、身份证号）
4. 选择脱敏模式
5. 点击"开始脱敏"
6. 下载脱敏后的文件和对照表

### 数据还原
1. 切换到"数据还原"标签
2. 上传脱敏后的 Excel 文件
3. 上传对照表
4. 点击"开始还原"
5. 下载还原后的原始文件

---

## 脱敏模式说明

| 模式 | 示例 | 说明 |
|------|------|------|
| 保留首尾 | `张三` → `张*三` | 适合姓名、手机号等 |
| 完全随机 | `张三` → `MASK_A3F7B2D1` | 完全替换为随机标识符 |

---

## 注意事项

1. **对照表很重要**：脱敏后生成的对照表用于还原，请妥善保管
2. **数据备份**：建议先备份原始数据再操作
3. **临时文件**：脱敏过程中产生的文件会保存在用户目录下的 `.excel_desensitizer` 文件夹
